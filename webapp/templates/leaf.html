This model attempts to identify tree genus from user provided pictures.
It is currently trained on 7 tree genus from the pacific northwest. These are:
Alder (genus Alnus)
False Cedar(genus Calocedrus, Thuja, Chamaecyparis)
Douglas Fir(genus Pseudotsuga)
True Fir (genus Abies)
Maple (genus Acer)
Oak (genus Quercus)
Pine (genus Pinus)
